The keynav Add-on

keynav is an add-on for Thunderbird that allows the folder pane to be quickly navigated by typing part or all of a folder name when it has the focus.

Note that single-letter Thunderbird short-cuts (such "n" and "p" which move to the next and previous unread messages respectively) do not work in the folder pane when key navigation is enabled. Therefore, There is an option to turn key navigation on/off, as well as for adding this item to the "Go" menu.

Where to Get keynav

A pre-built binary of keynav can be downloaded by clicking on the download link below. See "How to install keynav" for instructions on how to install the xpi file.

Download keynav

The source code for keynav is available on gitHub. Alternatively, the git repository can be cloned directly at the command line by typing
git clone https://github.com/hartag/keynav.git keynav
which places a copy of the repository in a directory called "keynav".

How to Install keynav

Once you have downloaded the xpi file, it can be installed using the following procedure.

  1. Open Thunderbird.
  2. Click on the Tools menu and then click on Add-ons to open the Add-ons Manager. From the keyboard, this can be done by typing Alt+t followed by a.
  3. Click on the "Tools for all add-ons" button, which appears as a cog wheel. It is just before the "Search all add-ons" edit field in the tab order.
    • Note to screen-reader users. The button needs to be clicked using the mouse by simulating a left mouse-button click at the location of the button on screen.

      First use Tab and Shift+Tab to move the focus to the "Tools for all add-ons" button.

      Next, you must move the mouse pointer to the button and then click the left mouse button to open the menu. The following table shows the commands for doing this in various screen readers.

      OperationDesktop CommandLaptop Command

  4. Click on the "Install Add-on from file" item in the pop-up menu that appears.
  5. An open file dialog wil appear. Use it to select the xpi file for the keynav add-on you downloaded and click the "Open" button.
  6. Follow the instructions to complete the installation. You may be asked to restart Thunderbird.

Building From Source

To build the add-on from source, first download the source code from GitHub. In addition to a shell such as bash, the following standard Linux/Unix utilities are needed for building the extension: make, cat, grep, head, sed, zip.

Building the extension is straightforward. After cloning the repository or downloading an archive of the repository and extracting it to a directory, simply type "make" at the command prompt to build the .xpi file.

In addition, "make showversion" will display the source code version and "make clean" will clean up the repository folder structure, deleting the zip file, as well as files with names like *.bak, etc.

Once the xpi file has been built, it can be installed using the instructions given above in How to Install keynav.
